# IP Info Tool
* `Just a simple tool to get IP's information, it uses 'requests'  module to gather information about IP. `

# Features
* `Country`
* `Zip Code`
* `Latitude`
* `Longitude`
* `Region`
* `ISP (Internet Service Provider)`
* `Proxy (True/False)`
* `Hosting (True/False)`
